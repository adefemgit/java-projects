Task 4 — Use Cases and User Stories

Use Cases (5)
1. Register and Login — actor: Learner
2. Browse Quiz Catalog — actor: Learner
3. Take Quiz with Randomized Questions — actor: Learner
4. Submit and View Results — actor: Learner
5. Review Answers and Explanations — actor: Learner

User Stories (5)
1. As a learner, I want to register/login so that I can access quizzes and track results.
2. As a learner, I want to filter quizzes so that I can find suitable content quickly.
3. As a learner, I want randomized questions/options so that cheating is reduced.
4. As a learner, I want instant scoring and breakdown so that I understand my performance.
5. As a learner, I want to review answers and explanations so that I learn from mistakes.
